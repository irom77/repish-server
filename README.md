# CheckPoint 1100 Appliance config generator
### INSTALLATION:
1. git pull https://github.com/irom77/repish-server.git
2. cd repish-server
2. bower install
3. npm install
4. forever start bin/www

### USE:
 **http://wwww:3001/**
